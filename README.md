## 📖 Описание проекта

**WpfApp_Krasotka** — это современное приложение с графическим интерфейсом на базе **WPF**, разработанное для нужд администраторов салона красоты.  
Приложение позволяет вести клиентскую базу, управлять услугами, расписанием мастеров и осуществлять запись клиентов, а также импортировать/экспортировать данные.

### 🌟 Основные возможности

*   👥 **Управление клиентами:** добавление, редактирование и хранение базы клиентов.
*   💇‍♀️ **Мастера и услуги:** учет квалификации мастеров, настройка стоимости услуг и их категорий.
*   📅 **Записи (Appointments):** удобный интерфейс записи клиентов на конкретное время к нужному мастеру.
*   ⚙️ **Работа с БД:** встроенный инструмент для настройки и проверки подключения к базе данных.
*   📊 **Интеграция с Excel:** формирование отчетов и экспорт данных (с использованием библиотеки EPPlus).

## 🗄️ Настройка базы данных (MySQL)

В корне проекта находится папка **`!База данных в MySQL`**, которая содержит все необходимые SQL-скрипты для быстрого развертывания базы данных с нуля.

Для корректной работы приложения необходимо:
1. Открыть ваш MySQL клиент (например, **MySQL Workbench** или **phpMyAdmin**).
2. Выполнить скрипты в следующем порядке:
   *   📄 `create database.sql` — создание самой структуры базы данных.
   *   📄 `all tables.sql` — создание необходимых таблиц (`Clients`, `Masters`, `Services` и т.д.).
   *   📄 `masters and clients and servType.sql` / `InsertTables.sql` — заполнение базы тестовыми/базовыми данными (справочники).
   *   *(Опционально)* Скрипты `UpdateServices.sql`, `DeleteServices.sql` и `services table use.sql` представлены как примеры запросов и операций с базой.

## 🚀 Запуск проекта

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/DimaPolezhaev/WpfApp_Krasotka.git
   ```
2. **Разверните базу данных** согласно [инструкции выше](#️-настройка-базы-данных-mysql).
3. **Откройте проект** `WpfApp_Krasotka.sln` в **Visual Studio 2022+**.
4. Обновите NuGet-пакеты (MySqlConnector, EPPlus), если это потребуется.
5. Запустите проект (`F5`).
6. В открывшемся приложении зайдите в раздел **"Работа с Базой Данных"** -> **"Открыть БД"** и укажите актуальные данные для подключения к вашему локальному или удаленному серверу MySQL.

