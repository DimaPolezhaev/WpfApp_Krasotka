# Салон красоты "Красотка" (WpfApp_Krasotka)

## Описание проекта
Данный проект представляет собой desktop-приложение для автоматизации работы салона красоты "Красотка" и управления его рабочими процессами. Приложение разработано с использованием фреймворка **WPF (Windows Presentation Foundation)** на платформе **.NET 8.0 (C#)**.

## Основной функционал
Проект условно разделен на два основных модуля (через главное окно):
1. **Работа с Базой Данных** — настройка и управление подключением к базе данных (через форму `OpenDB`).
2. **Ежедневные функции (Администратор салона)** — рабочее пространство администратора для ведения записей и учета.

### Управление сущностями
В приложении реализован функционал для управления следующими справочниками и процессами:
- **Clients (Клиенты):** база данных клиентов салона.
- **Masters (Мастера):** информация о сотрудниках и их расписании.
- **ServTypes (Категории услуг):** классификация оказываемых услуг.
- **Services (Услуги):** конкретные процедуры салона с указанием стоимости и других параметров.
- **Appointments (Записи):** планирование и отображение записей клиентов к мастерам.

## Технологический стек
- **Язык программирования:** C#
- **Платформа:** .NET 8.0, WPF (Windows)
- **Работа с СУБД:** MySQL (через пакет `MySqlConnector` 2.4.0)
- **Работа с Excel:** `EPPlus` 7.0.0 (используется для формирования отчетов, импорта или экспорта данных в таблицы).

## Структура проекта
- `MainWindow.xaml` — главное меню приложения.
- `OpenDB.xaml` — окно подключения и базовой конфигурации БД.
- `EveryDay.xaml` — основное рабочее окно администратора для повседневных задач.
- Интерфейсы разделов данных: `Clients.xaml`, `Masters.xaml`, `Services.xaml`, `ServTypes.xaml`, `Appointments.xaml`.

## Запуск проекта
1. Откройте файл решения `WpfApp_Krasotka.sln` в IDE (рекомендуется Visual Studio 2022).
2. Выполните восстановление пакетов NuGet (Project -> Manage NuGet Packages -> Restore).
3. При необходимости разверните дамп базы данных MySQL.
4. Настройте параметры подключения к БД внутри приложения.
5. Закомпилируйте и запустите приложение (горячая клавиша `F5`).
